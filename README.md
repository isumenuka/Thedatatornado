
# The Data Tornado (Config Makeathon)

This is the code bundle for **The Data Tornado** (originally *Display NOAA Datasets*).

For the complete design workflow, technology stack details, and submission guidelines regarding Figma's AI suite, please see the [Config Makeathon Submission Guide](file:///c:/Users/Isum/Documents/TheDataTornado/MAKEATHON_SUBMISSION.md).

## Running the Code

1. Install the dependencies:
   ```bash
   npm install
   ```

2. Start the local development server:
   ```bash
   npm run dev
   ```

3. Build the production bundle:
   ```bash
   npm run build
   ```
